Данный проект представляет собой распределенную систему, объединяющую высокопроизводительные сервисы на **Go** и клиентскую оркестрацию на **Python**. Проект разработан с использованием принципов **Agentic Engineering** (инструмент Claude Code).

## Обзор системы

Программа реализует мультиязычную архитектуру, где критические узлы (безопасность, вычисления, сеть) вынесены в Go-сервисы, а бизнес-логика и тестирование реализованы на Python.

### Основные возможности (Вариант 7):
* **REST API (Go & Gin):** Высокопроизводительный веб-сервер с валидацией данных.
* **gRPC Service:** Бинарный протокол взаимодействия между Go-сервером и Python-клиентом.
* **JWT Security:** Защита эндпоинтов с использованием токенов (HS256).
* **Graceful Shutdown:** Безопасное завершение процессов Go при получении системных сигналов.
* **Python Integration:** Автоматизированные клиенты для взаимодействия с микросервисами.

## 🛠 Технологический стек

* **Languages:** Go 1.21+, Python 3.10+
* **Go Frameworks:** [Gin Gonic](https://gin-gonic.com/), [Validator v10](https://github.com/go-playground/validator)

## Сборка и запуск

Проект разделен на независимые модули согласно шагам лабораторной работы. Перед запуском убедитесь, что у вас установлены **Go 1.21+** и **Python 3.10+**.

### 1. Настройка окружения Python
Рекомендуется использовать виртуальное окружение для изоляции зависимостей:
```bash
python -m venv venv
source venv/bin/activate  # Для Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### 2. Запуск Go сервисов (REST / gRPC)
#### Пример для gRPC сервера
```
cd задание_повышенной_сложности_1/go_service
go mod tidy
go run main.go
```
### 

### 3. Запуск Python-клиентов
Клиенты взаимодействуют с запущенными Go-сервисами. Откройте новый терминал и выполните:
```
# Активируйте venv, если не сделали этого ранее
source venv/bin/activate

# Запуск gRPC клиента
python "задание_повышенной_сложности_1/python_client/client.py"

# Запуск теста JWT авторизации
python "задание_повышенной_сложности_3/python_client/test_jwt_verifier.py"
```
